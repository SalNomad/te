# EU Tender Match

Static website for `eutendermatch.com`, ready for GitHub Pages.

## Main offer

EU Tender Match emails TED tender reports that match a company's services, countries, markets, CPV codes, and bidding needs.

## Launch offer

Selected founding customers can request the first week free. Paid plans are simple 30-day launch plans: Daily Report €149 / 30 days, Weekly Report €79 / 30 days, Monthly Report €39 / 30 days. No yearly contract.

## Publish on GitHub Pages

1. Create a GitHub repository, for example `eutendermatch-website`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Add your custom domain: `eutendermatch.com`.
7. Configure your DNS records at your domain registrar.

## Before launch

- Replace `[DATE]`, `[COUNTRY]`, `[CITY/COUNTRY]`, and `[COMPANY LEGAL NAME]`.
- Create real inboxes or forwarding aliases for:
  - hello@eutendermatch.com
  - support@eutendermatch.com
  - billing@eutendermatch.com
  - legal@eutendermatch.com
  - privacy@eutendermatch.com
- The inquiry form currently uses `mailto:`. For production, connect a real form backend such as Formspree, Netlify Forms, Basin, Tally, Typeform, or your own backend.
- Ask a qualified lawyer to review the legal pages before commercial launch.
