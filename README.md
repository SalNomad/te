# EU Tender Match

Static website for `eutendermatch.com`, ready for GitHub Pages.

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Add your custom domain: `eutendermatch.com`.
7. In your domain registrar DNS, add the GitHub Pages DNS records.

## Before launch

- Replace `[DATE]`, `[COUNTRY]`, `[CITY/COUNTRY]`, and `[COMPANY LEGAL NAME]`.
- Replace contact emails with real inboxes.
- Connect the inquiry form to a real form backend if you do not want mailto.
- Ask a qualified lawyer to review legal pages before commercial launch.
